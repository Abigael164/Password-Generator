#  Password Generator App

A simple and customizable password generator built using React.  
The app allows users to generate strong, random passwords based on selected criteria such as length, numbers, symbols, and uppercase letters.

---

##  Features
- Generate a random secure password
- Choose password length
- Include/exclude:
  - Numbers
  - Symbols
  - Uppercase letters
  - Lowercase letters
- Copy password to clipboard with one click
- Fully responsive UI
- Real-time password updates

---

##  Technologies Used
- React (Vite)
- JavaScript (ES6)
- CSS
